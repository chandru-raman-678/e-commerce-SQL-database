# E-Commerce SQL Database (MS SQL)

## Project Details: 
This is an MS SQL real-time work, and it contains DDL, DML, DQL, TCL queries for creating tables, inserting data, committing written queries, and data retrieval. Here, indexing was used for efficient data retrieval and improved performance by 30% when creating the index. Used sample data for the entire task; moreover, it can be used in this e-commerce environment for all kinds of huge data. Here, Triggers are used for tracking updates such as insert, delete, or other activities in orders or catalogs.
View is used for simplifying, securing, and reusing data queries, so it can save time because if you write a huge query need more time to write, and each time that same query would write everywhere and can hide the actual query from unknown users sometimes.
Stored procedures and Functions are used for mathematical calculation while get fast results. These are representing overall entire work and its purpose.
